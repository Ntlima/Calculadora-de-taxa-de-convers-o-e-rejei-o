 📊 Calculadora de Taxa de Conversão e Rejeição

Uma aplicação web simples e funcional para análise de métricas digitais, permitindo comparar dados de conversão e rejeição de forma estratégica.

 🚀 Funcionalidades

- 📈 Cálculo de taxa de conversão
- 📉 Cálculo de taxa de rejeição
- 🔄 Comparação Month over Month (MoM)
- 📊 Comparação Year over Year (YoY)
- 💻 Interface responsiva e intuitiva

🛠️Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript  
- Bootstrap  

💡 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar lógica de programação, manipulação de dados e construção de interfaces modernas voltadas para análise de métricas.

A aplicação permite que usuários insiram dados de diferentes períodos e obtenham rapidamente insights sobre variações percentuais.
